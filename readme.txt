== Description ==

Kurz-URL Dienst auf Basis von <a href="http://yourls.org/" title="Your Own URL Shortener">YOURLS</a>. Erstellt automatisch eine Kurz-URL und teilt diese mit dem Titel in Deinem Twitter-Konto.
